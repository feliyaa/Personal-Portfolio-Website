BTCUSD 1-Minute Prediction Model

A high-frequency machine learning project designed to predict short-term Bitcoin price movements using market indicators.

Back to Portfolio

This project explores rapid market prediction through feature-driven modelling. By combining Polygon.io data, technical indicators, and time series analysis, it aims to forecast 1-minute BTCUSD price direction with a focus on reliability and early signal detection.

  • Model type: time series / classification
  • Data source: Polygon.io crypto market data
  • Performance: up to 70% sample accuracy
  • Tools: Python, pandas, technical indicators, model evaluation
  • Goal: practical insight generation for fast market scenarios
